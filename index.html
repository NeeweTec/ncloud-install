Ncloud Agent

Gerencie serviços TOTVS Protheus com facilidade. Um comando para instalar, configurar e monitorar.

Instalação
$ curl -fsSL https://get.neewecloud.com/install.sh | sudo bash

Auto-discovery

Detecta automaticamente ambientes Protheus no servidor

Monitoramento

Status em tempo real de todos os serviços e processos

Webhooks

Notificações automáticas quando serviços mudam de estado

API REST

Integração completa com sistemas externos

Seguro

Autenticação via token e comunicação criptografada

Ubuntu, Debian, CentOS

Suporte às principais distribuições Linux

CLI Simples

Comandos diretos para gerenciar o agent

ncloud start Iniciar
ncloud stop Parar
ncloud restart Reiniciar
ncloud status Ver status
ncloud logs Logs em tempo real
ncloud menu CLI interativo
